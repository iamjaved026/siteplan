# MetroConstruct Project Blueprint

## Overview

This document outlines the architecture, features, and development plan for the MetroConstruct project management dashboard. The application is built using Next.js with the App Router and is designed to provide a comprehensive, real-time overview of a complex construction project.

## Core Architecture

*   **Framework:** Next.js (App Router)
*   **Styling:** CSS Modules and global styles
*   **Deployment:** Firebase Hosting

## Features

*   **Dashboard:** The main dashboard provides a high-level overview of the project status.
*   **KPI Cards:** The dashboard includes KPI cards that display key project metrics.
*   **Gantt Chart:** The dashboard includes a Gantt chart that displays the project timeline.
*   **Alerts:** The dashboard includes an alerts section that displays important project notifications.
*   **Weather Forecast:** The dashboard includes a weather forecast section that displays the 7-day forecast.
*   **Task Cards:** The dashboard includes a task cards section that displays active tasks.
*   **Time Prediction:** The dashboard includes a time prediction chart that displays the predicted project completion time.
*   **Resource Availability:** The dashboard includes a resource availability section that displays the availability of project resources.
*   **Budget vs. Actual:** The dashboard includes a budget vs. actual section that displays the project budget and actual spending.
